# simpExcelPaser

> 개요 : 동료직원이 Excel 파일에 json형식을 파싱하는 도구를 만들어 달라고해서 제작 

### Spac : JavaScript(ES6+),Electron,npm

- ##### dev mode Set

  1. git clone
  2. console edit  "npm i" 
  3. finish
 
- ##### run app
  console edit 
  <pre>
    npm start
  </pre>
  
  
- ##### build app
  
  * 요구사항 
    * windows : python 2.7+ , python X 
                Visual Studio Build Tools 설치 필요
   
   * Mac : 필요 없음
   
   * 도구 : electron-builder
   
   <pre>
    npm run build: "OS"
   </pre>
